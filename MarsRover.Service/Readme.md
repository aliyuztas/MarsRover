# Mars Rover

Mars Rover, a project which explores some spesific areas by moving with instructions from defined starting positions by two robots who are used to explore a matrix area in Mars.

## Projects Technical Information

This projects is developed with C# in .NET 5 framework. Unit Tests are developed with NUnit package.
Mars Rover includes two seprate projects;

```bash
MarsRover.Service
MarsRover.Test
```

#### MarsRover.Service; 
This projects includes the necessary algorithm and the logical operations to be able to do the main work. There are interfaces, services, models, enums and costants included.
#### MarsRover.Test;
This project is developed for MarsRover.Service project's unit testing purposes.
